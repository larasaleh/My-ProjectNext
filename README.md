1- What is MCP?

 Project Purpose
This project is part of the NextFlows Academy Week 1 assignment.
It introduces the basics of Git, GitHub, and the Model Context Protocol (MCP).

101 Requirements
- Git installed
- GitHub account
- Visual Studio Code

2 How to Run the Project


1. Check that Git is installed:

   git --version
   

3. Configure Git:
   bash
   git config --global user.name "Your Name"
   git config --global user.email "your@email.com"
  

4. Create a new repository on GitHub.

5. Clone the repository:
  bash
   git clone <repository-url>
   

6. Open the project folder:
  bash
   cd <repository-name>
   

7. Edit the `README.md` file.

8. Save your changes:
 bash
   git add .
   git commit -m "Update README"
   git push
